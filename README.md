### ✅ 1. Mac 에서 node.js 설치하기!
* Node.js®는 Chrome V8 JavaScript 엔진 으로 빌드된 JavaScript 이다.
* 설치주소 : 
https://nodejs.org/ko

### ✅ 2. 터미널로 설치하기
#### Homebrew 설치 (https://brew.sh/index_ko)

	/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"		

#### node.js 설치

	$ brew install node


#### node.js 설치버전 확인하기
	$ node -v
